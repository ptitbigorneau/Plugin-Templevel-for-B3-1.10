# templevel plugin

# Plugin for B3 (www.bigbrotherbot.net)

# PtitBigorneau - www.ptitbigorneau.fr


templevel plugin (v1.4) for B3

Requirements
------------

* BigBortherBot(3) >= version 1.10

Installation

------------

 
* copy templevel folder into b3/extplugins
 
* copy templevel.ini into b3/extplugins/conf
 
* create the templevel table in your database importing the tlevel.sql file.
 
* update your main b3 config file with :
 <plugin name="templevel" config="@b3/extplugins/conf/templevel.ini"/>
